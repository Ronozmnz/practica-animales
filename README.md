# 🦁 Proyecto: Clasificación de los Animales

¡Bienvenido! Este es un sitio web interactivo y responsivo diseñado para explorar el fascinante mundo de la fauna, dividida según su estructura ósea en **Vertebrados** e **Invertebrados**.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica de las páginas (`index.html`, `vertebrados.html`, etc.).
* **CSS3:** Diseño personalizado, diseño responsivo usando *breakpoints* y estilos visuales.
* **Git & GitHub:** Control de versiones y despliegue en la nube.

---

## 🚀 Características del Sitio

* 📱 **Diseño Adaptable:** Optimizado para computadoras, tablets y teléfonos celulares.
* 🎨 **Interfaz Moderna:** Tarjetas visuales para una navegación intuitiva.
* 🧭 **Menú de Navegación:** Acceso rápido a las secciones de Vertebrados, Invertebrados y Formulario de Contacto.
